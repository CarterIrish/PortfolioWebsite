Too the left of this div element is another div element. In the other div element there exists an iframe element. The iframe element in html allows one webpage to embed another webpage within itself. It is not always an option however, if the website contains an X-Frame-Options header. This header is used as a security measure by browsers to stop click-jacking attemts on websites. essentially preventing them from being embeded into a malicious webpage that wants you steal information from you.
